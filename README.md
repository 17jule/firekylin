# FireKylin

A Simple & Fast Node Bloging Platform Base On ThinkJS 2.0 & ReactJS & ES2015


## install

```sh
git clone https://github.com/75team/thinkjs.git
```

## install dependencies

```js
npm install
```

## compile

```js
npm run compile
```

## start server

```js
npm start
```

* home <http://localhost:3000/>
* admin <http://localhost:3000/admin/>
